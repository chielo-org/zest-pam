# `zest-pam`

Another wrapper of PAM (Pluggable Authentication Modules).

## Credits

### `zest-pam-core`

This crate is a fork of
[`pam-sys2` by Eddie BILLOIR](https://github.com/LeChatP/pam-sys2), which is a
fork of [`pam-sys` by Florian Wilkens](https://github.com/1wilkens/pam-sys).

### `zest-pam-module`

This crate is a fork of
[`nonstick` by Paul Fisher](https://github.com/thetorpedodog/pam-rs-dev), which
is a fork of [`pam-rs` by Anthony Nowell](https://github.com/anowell/pam-rs).

`pam-rs` was in turn inspired by:

- [`rust-pam` by tozny](https://github.com/tozny/rust-pam)
- [`pam_groupmap` by ndenev](https://github.com/ndenev/pam_groupmap)
- [`pam-http` by beatgammit](https://github.com/beatgammit/pam-http)
